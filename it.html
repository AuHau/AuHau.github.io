Home Myo

Android
Java
Smart Homes
Myo armband
Home Myo is proof-of-concept project of advanced Smart Home control using hand gestures. It was developed as part of my Bachelor thesis Intelligent Building Control by Hand Gestures. It utilizes Myo armband with its hand gesture detection capabilities and in-house developed basic indoor localization.

It is Android application, which connects several components into whole system. User can configure and map the hand gestures to the actions inside Smart Home. Then application listen for the Myo detected hand gestures and then based on the user location inside the Smart Home, it sends proper command into Smart Home network.

Select Picker

JavaScript
jQuery
Plugin
Select Picker is jQuery plugin supporting work with select boxes. It extends default possibilities of select boxes to new range of features. In basic set up, it will simply mimic basic normal select box, but in the same time provide you extensive ways of styling the appearance of your select box and several more neat features.

Main field where Select Picker truly rocks is when you want to use multiple-selection select boxes. It will provide you easy way how to facilitate user's selections with tag-like appearance solution, providing you features like custom styling and search.
